### 前端：
先去下好node（node现在自带npm）和vue3（直接csdn一步一步来），然后打开项目后在终端根目录输入npm i，然后npm run dev 。
### 后端：
根目录下的cdr-backend文件夹是后端，cd到这个文件夹之后node server.js
